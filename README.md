# kees

I developed this application for fun between me and my friends this application is in Arabic so the application for managing and tracking "كيس" (bags) means lies with associated details such as the number of sacks, lie details, and other related information. The app has multiple screens, and it utilizes Flutter's Material Design components for the user interface.
